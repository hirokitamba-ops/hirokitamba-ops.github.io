Abstract

This paper defines the TLIM Research Program as a program-level scholarly framework centered on the analysis of three interacting layers: text, ritual, and genealogy. It serves as the foundational node for the broader publication architecture by linking the core definition of TLIM to related papers, preprints, and subsequent outputs. Within this website, the page functions not only as an individual paper landing page but also as the primary structural hub for the wider research program.

Program-level role

This page is the anchor node of the TLIM Research Program. It provides the foundational definition of the framework and connects outward to the papers below. Each linked paper should, in turn, link back to this page as the shared program reference.
